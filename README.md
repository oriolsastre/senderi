# Excursions

Page to handle, store and show my hikes. Simple login to be able to create and update hikes.

- Frontend with vite
- Backend with express and linked to the front with vite-express
- Sqlite for Database
- Simple JWT for authentication.

## Main Page
Main page listing all the available hikes ordered by date

## Hike Page
Each hike has it's page. Showing

- Title
- Map: gpx track is stored in openstreetmap and fetched from there
- Description
- Pictures fetched from an API.
